प्रोग्रामिंग — भाग ४
C++ में सरलतम प्रोग्राम बनाना
Visual Studio 2022 स्टार्ट करें；
> Create a new project
> Console App
(Run code in a Windows terminal. Prints "Hello World" by default)
यदि “Console App” सूची में न दिखे तो सर्च−बॉक्स में Console App लिखें ।
तब ConsoleApplication1 नाम से नया एप्प बनने का ऑप्शन आयेगा，उसका नाम छोटा कर दें ＝ ConsoleApp1，और पथ मनचाहा चुन लें，जैसे कि Z:\CPP ॥
उस पथ में पूरा एप्प बन जायगा । उसमें Console App.cpp नाम का फाइल दिखेगा । ऊपर Local Windows Debugger को क्लिक करने पर एप्प रन होगा और Windows terminal अर्थात् Console पर "Hello World" लिखा मिलेगा । उसे बन्द करें ।
Console App.cpp में सम्पूर्ण प्रोग्राम है＝
#include <iostream>
int main()
{
std::cout << "Hello World!\n";
}
#include <iostream> के कारण Console पर आउटपुट std::cout भेजा जा सकता है । अब Console App.cpp में समस्त सामग्री को हटाकर निम्न पेस्ट कर दें⋅
// के पश्चात जो कुछ लिखा रहेगा वह प्रोग्राम का भाग नहीं，केवल टिप्पणियाँ हैं जो प्रोग्राम को समझने के लिए आप लिख सकते हैं अथवा हटा सकते हैं ।
iomanip के कारण कितने अङ्कों की गणनात्मक शुद्धि चाहिए इसे आप सुनिश्चित कर सकते हैं । इस कोड में मैंने शुद्धि ＝ ६० की है जिसे आप बदल सकते हैं ।
═══════════════════════════
// ConsoleApp.cpp : This file contains the 'main' function. Program execution begins and ends there.
//
#include <iostream>
#include <iomanip> // for std::setprecision
int main() {
long double pi = 3.141592653589793238462643383279502884197169399375105820974944592307816406;
long double y = pi / 2.2;
long double result = pi * y;
std::cout << std::setprecision(60); // Set precision to 60 digits
std::cout << "The result is " << result << "\n";
return 0;
}
// Run program: Ctrl + F5 or Debug > Start Without Debugging menu
// Debug program: F5 or Debug > Start Debugging menu

// 6. In the future, to open this project again, go to File > Open > Project and select the .sln file
═══════════════════════════
इसे रन करेंगे तो pi * y का गुणनफल result टर्मिनल पर दिखेगा ।
#include <iomanip> को हटा देंगे तो std::cout << std::setprecision(60); भी हटाना पड़ेगा，तब long double के कारण गणनात्मक शुद्धि १९ अङ्कों की मिलेगी । सभी स्थानों पर long double के बदले केवल double रखेंगे तो शुद्धि १६ अङ्कों की मिलेगी । double के बदले अन्य डैटा टाइप बिंग कोपायलट से पूछकर सीख लें । तब सरल प्रोग्राम होगा＝
═══════════════════════════
#include <iostream>
int main() {
double pi = 3.141592653589793;
double y = pi / 2.2;
double result = pi * y;
std::cout << "The result is " << result << "\n";
return 0;
}
═══════════════════════════
int main() के पूरे कोड को आप बिंग कोपॉयलट की सहायता से बदल सकते हैं । जैसे कि उससे पूछें कि अमुक गणित का कोड दें ।
Project > Property > C/C++ > Output Files > Assembler Output > Assembly-Only Listing(/FA) कर देंगे तो असेम्बली लैंग्विज में पूरे प्रोग्राम का कोड .asm एक्सटेंशन के फाइल में देगा，और यदि Assembly With Source Code (/FAs) चुनेंगे तो उस .asm फाइल में असेम्बली लैंग्विज के साथ C++ का सोर्स कोड भी रहेगा जिसकी सहायता से असेम्बली लैंग्विज सीख सकेंगे । आरम्भ में .asm एक्सटेंशन वाला यह फाइलन भयानक लगेगा，किन्तु Bing Copilot, chatGpt और Google Search की सहायता से Assembly Language सीखना चाहेंगे तो यह .asm फाइल बहुत काम देगा । किन्तु नौसिखियों को Assembly Language से दूर ही रहना चाहिए，C++ की तुलना में Assembly Language अधिक तीव्रता से तभी कार्य कर सकेगी जब आप उसमें दक्ष हों，वरना सामान्यतः C++ की गति भी कम नहीं रहेगी ।
मेरी राय है कि असेम्बली लैंग्विज न सीखें और Project > Property > C/C++ > Output Files को खाली रहने दें । भविष्य में कभी काम दे यही सोचकर मैंने तरीका बता दिया है । MASM में ३२ बिट और NASM में ६४ बिट का असेम्बली लैंग्विज में सम्पूर्ण प्रोग्राम बना सकते हैं ।
